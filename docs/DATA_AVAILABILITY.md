# Data availability

This release contains the input tables, archived output tables, metadata, and source code used for the Gaia--HST profile-level validation workflow.

The machine-readable output tables are provided in `output_csv/`. Input tables and raw/processed HST/HACKS support files are provided in `input/`. Source provenance is documented in `metadata/source_provenance.csv`.

Third-party data products retain their original acknowledgement and citation requirements. Users reusing the data should cite the original Gaia, HST/HACKS, and APOGEE/SDSS sources in addition to citing this repository release DOI.
