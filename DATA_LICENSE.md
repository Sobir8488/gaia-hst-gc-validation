# Data licensing note

The MATLAB source code is released under the MIT License in `LICENSE`.

Workflow-created metadata and derived output tables in `metadata/` and `output_csv/` may be cited through the repository DOI. Third-party Gaia, HST/HACKS, and APOGEE/SDSS data layers retain their original provider terms, citation requirements, and acknowledgement requirements.

Before public repository release, confirm that each included third-party input table can be redistributed under the intended repository license. If redistribution is not permitted for any input file, remove that file and keep only the source URL, checksum of the locally used file, and the processing script needed to reconstruct it.
